# 그냥 알파벳 옮기는 사이트

Vue를 공부하기 위해 진행한 프로젝트<br>
<br>

[그냥 알파벳 옮기는 사이트](http://jma.cool)

<br>
로그인을 하고 알파벳을 드래그해서 옮기는 사이트. 로그인의 경우 페이스북만 된다. <br>
<br>
다른사람이 옮기는 알파벳을 실시간으로 확인할수 있으며 '실시간 업데이트' 이는 파이어베이스의 firestore가 제공하는 기능이다. <br>
